Clocked Time Tracking Software

Time tracking software can help you become more efficient and productive. Clocked helps you monitor time spent on each project or individual tasks.

The server side of Clocked was created with python and Django and, GraphQL for creating the api endpoints, and HTML + CSS, React.js for the client side, with other javascript frameworks that really made my work developing this app as a sole developer much simpler.

Check it out here!

I-shop e-Commerce Website

An eCommerce web app that allows the users to buy items online and the items be shipped to their address if items are not digital. This app has a special feature that uses cookies to allow non logged in users to purchase items without singing in. Made with django, javascript, MDbootstrap.

Invoice Manager

This app was made for a company that requested a specific invoice app that allows the users to create a company profile and customers to generate invoices to. Made with Python, Django, PostgreSql, Heroku.

Stock management

A prototype stock management app that allows the user to record stock into the database. The app consist of a search function with the ability to store the search results into a csv file, a history page that records the updates made on every item by the user. Made with django, MySql, Jquery, Bootstrap.

Da Vinci Art Grid

As an artist we struggle drawing objects in the correct proportions. This app allows users to upload an image which will therefore be rendered with grids to assist the artist in drawing the object with accurate proportions. Made using python, django and pillow library.
